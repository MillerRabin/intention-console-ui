Распознавание голоса в ваших проектах

Вы можете использовать консоль для того, чтобы обеспечить распознавание голоса в ваших проектах

Откройте вкладку Намерения. Найдите намерение "Забираю пользовательский ввод с микрофона или клавиатуры" с ключем HTMLTextAreaElement - Recognition. Выходной ключ Recognition означает, что намерение выдает голосовой ввод в виде текста.
HTMLTextAreaElement - означает, что также передается ссылка на HTML элемент textarea, чтобы принимать ввод с клавиатура, но ее можно не передавать
Это намерение было разработано, чтобы обеспечить передачу данных внутри консоли, но мы без труда можем использовать его из внешних проектов.
Чтобы подключиться к этому намерению - нам нужно создать встречное намерение с противоположным ключем в нашем проекте.
Возьмите шаблон main.js созданный в разделе Создание приложения
Допишите в конец файла следующий код

main.js intentionStorage.createIntention({
  title: {
    en: 'Recognition test',
    ru: 'Тест распознавания'
  },
  input: 'Recognition',
    output: 'HTMLTextAreaElement',
    onData: async function onData(status, intention, value) {
    if (status != 'data') return;
    console.log(value);
  }
});

Запустите проект

node main.js

Как только ваше хранилище снова окажется в статусе online все, что вы говорите в микрофон или пишете в окне пользовательского ввода, выведется в консоль вашего проекта.
